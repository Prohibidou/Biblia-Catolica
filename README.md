# Biblia Católica

Complete Catholic Bible reader with Navarra Bible commentary.

## Features
- Complete Catholic Bible (73 books)
- Navarra Bible commentary (3,250+ comments)
- Section titles (739 titles)
- 35,855 verses total

## Live Site
https://bibliacatolica.netlify.app

## Source
PDF: https://drive.google.com/file/d/1r0HUELJztDNZIPypfOUKKF87nkCIM0tU/view

## Tech Stack
- React + TypeScript
- Vite
- SQLite (SQL.js)
- Netlify (auto-deploy from GitHub)

## Database
- Old Testament: 27,949 verses, 2,571 comments
- New Testament: 7,906 verses, 679 comments
